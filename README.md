# SidraFacil
Pacote R, para extração automatizada e simplificada de dados do Sistema IBGE de Recuperação Automática - SIDRA. Ao instalar o pacote e chamá-lo, o usuário apenas precisará digitar o número da tabela desejada. Arquivo salvo em pasta indicada pelo usuário conterá os dados disponíveis na tabela. Ao final será salvo arquivo com os dados na pasta.
